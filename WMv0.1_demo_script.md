# Script vidéo démo — World Model v0.1
**Durée cible : 3 minutes | Format : screencast**

---

## Structure (3 actes)

### Acte 1 — Le problème (45s)

> *Ouvrir la page "Vue d'ensemble"*

"Nous sommes en 2026. 7 des 9 limites planétaires sont dépassées.
La question n'est plus *si* un collapse est possible — les modèles le montrent depuis 1972.
La question est : *quelle trajectoire choisissons-nous ?*"

> *Montrer les 4 métriques en haut (limites dépassées, population, ressources, HDI)*

"World Model v0.1 simule ces trajectoires sur un siècle."

> *Faire défiler le dashboard 4 variables*

"Trois courbes. Trois futurs possibles."

---

### Acte 2 — La simulation (1min 30s)

> *Aller sur "Scénarios" → sélectionner "Ressources"*

"Scénario BAU — Business as Usual. Les ressources s'épuisent progressivement.
En 2100 : 16% du stock initial. Le capital s'effondre. L'HDI plonge à 0.21."

> *Changer pour "Population"*

"La population plafonne puis décline. Pas par choix — par contrainte."

> *Changer pour "HDI" — montrer les 3 courbes ensemble*

"Scénario SW — Sustainable World. Même population finale.
Mais HDI à 0.86. Ressources à 70%. La différence ? Les choix faits **maintenant**,
dans la fenêtre 2025-2035."

> *Aller sur "Limites planétaires" → montrer le radar*

"Ces 9 limites définissent l'espace opératoire sûr pour la civilisation.
4 sont en zone critique. Le modèle simule les conséquences dynamiques de ces dépassements."

---

### Acte 3 — L'IA (45s)

> *Aller sur "Analyse IA" → sélectionner BAU → cliquer Analyser*

"Chaque scénario peut être analysé par Claude.
L'IA lit les trajectoires, identifie les mécanismes, propose des leviers d'action."

> *Laisser la réponse s'afficher*

"Ce n'est pas une prédiction. C'est un outil de raisonnement systémique —
pour comprendre les boucles de rétroaction, visualiser les bifurcations,
et orienter les décisions dans la fenêtre critique qui s'ouvre."

> *Revenir sur le dashboard*

"World Model v0.1. Prototype open-source.
Prochaine étape : intégration données temps réel, agents IA, MCP."
